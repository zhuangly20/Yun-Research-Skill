# arXiv 学术日报生成器

帮你把指定主题和时间范围的 arXiv 论文整理成中文日报，或针对具体问题在arXiv上进行学术调研。

## 功能

- **日报模式**：按学科分类浏览最新论文，生成精选日报
- **调研模式**：针对具体问题进行学术调研，用论文回答你的问题

## 支持平台

支持所有主流agent平台，以下是作者常用：
- AutoClaw（速度最快）
- Claude code搭配国产模型平替（需要自己配置api，内容质量最佳）
- Trae（完全免费）
